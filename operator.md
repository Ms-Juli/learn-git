# Операторы:
```
cat - вывести содержимое файла на экран 
nano - редактор файлов (nano README.md - открытие файла для редактирования)
cd - переходы по каталогам (cd .. - вернуться один уровень)
mkdir - создать папку в текущем каталоге
touch - создать пустой файл
pwd - команда, чтобы узнаь в каком каталоге находишься 
```
